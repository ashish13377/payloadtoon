# Code of Conduct 🤝

## Our Pledge

We want PayloadTOON to be a welcoming and respectful open-source project.

Everyone is welcome to participate regardless of background, experience level, or location.

## Expected Behavior

- Be respectful
- Be constructive
- Help others learn
- Give clear feedback
- Assume good intent

## Unacceptable Behavior

- Harassment
- Personal attacks
- Discrimination
- Spam
- Sharing private information without permission

## Enforcement

Project maintainers may remove comments, close issues, or restrict participation if someone violates this code of conduct.
